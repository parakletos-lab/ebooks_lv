# ebooks.lv Admin Hub

Эта страница — центральный навигационный хаб для административных инструментов ebooks.lv.

- Хаб-страница: [/admin/ebookslv/](/admin/ebookslv/)

В этом документе кратко объясняется назначение каждой карточки и указано, на какую страницу она ведёт.

---

## 1) Карточки на странице [/admin/ebookslv/](/admin/ebookslv/)

### 1.1 Заказы
- Открывает: [/admin/ebookslv/orders/](/admin/ebookslv/orders/)
- Описание: управляет доступом клиентов — импортирует оплаченные заказы из Mozello, позволяет создавать заказы вручную и связывает их с пользователями Calibre-Web, чтобы покупатели могли получить доступ к своим книгам.

### 1.2 Интеграция с Mozello
- Открывает: [/admin/ebookslv/mozello/](/admin/ebookslv/mozello/)
- Описание: настройка и контроль интеграции с Mozello (параметры API и webhook), обеспечивающие корректную обработку заказов и работу ссылок на продукты.

### 1.3 Книги
- Открывает: [/admin/ebookslv/books/](/admin/ebookslv/books/)
- Описание: синхронизация книг из библиотеки Calibre с продуктами Mozello (загрузка продуктов, экспорт и обновление, синхронизация цен и обслуживание каталога).

### 1.4 Шаблоны писем
- Открывает: [/admin/ebookslv/email-templates/](/admin/ebookslv/email-templates/)
- Описание: редактирование шаблонов исходящих писем ebooks.lv (например, писем о покупке и входе в систему), а также управление списком разрешённых токенов для каждого шаблона.

---

## 2) Кнопка на хаб-странице

### 2.1 Установить настройки по умолчанию
- Расположение: [/admin/ebookslv/](/admin/ebookslv/)
- Назначение: применяет рекомендованные настройки Calibre-Web по умолчанию (роли пользователей, боковая панель, заголовок страницы).
