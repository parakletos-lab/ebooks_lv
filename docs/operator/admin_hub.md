# ebooks.lv Admin Hub

This page is the central navigation hub for ebooks.lv admin tools.

- Hub page: [/admin/ebookslv/](/admin/ebookslv/)

This document explains what each card is and where it leads.

---

## 1) Cards on [/admin/ebookslv/](/admin/ebookslv/)

### 1.1 Orders
- Opens: [/admin/ebookslv/orders/](/admin/ebookslv/orders/)
- What it does: manage customer access by importing paid Mozello orders, creating manual orders when needed, and linking orders to Calibre-Web users so buyers can read their books.

### 1.2 Mozello
- Opens: [/admin/ebookslv/mozello/](/admin/ebookslv/mozello/)
- What it does: configure and monitor Mozello integration (API/webhook-related settings) so orders and product links work correctly.

### 1.3 Books
- Opens: [/admin/ebookslv/books/](/admin/ebookslv/books/)
- What it does: sync Calibre library books to Mozello products (load products, export/update a product, sync/push pricing, and handle product maintenance actions).

### 1.4 Email Templates
- Opens: [/admin/ebookslv/email-templates/](/admin/ebookslv/email-templates/)
- What it does: edit the outbound email templates used by ebooks.lv (e.g., purchase/login messaging), including the allowed tokens per template.

---

## 2) Button on the hub

### 2.1 Set default settings
- Button location: [/admin/ebookslv/](/admin/ebookslv/)
- Purpose: applies curated default Calibre-Web settings (roles, sidebar, title).
