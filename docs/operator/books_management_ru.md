# Управление книгами (Оператор)

Это упрощённая инструкция для повседневного управления книгами **ebooks.lv**.

Вы будете работать в двух местах:

- **Calibre** – здесь хранятся файлы книг и метаданные.
- **ebooks.lv Admin → Books Sync** ([/admin/ebookslv/books/](/admin/ebookslv/books/)) – здесь выполняется синхронизация с Mozello.

---

## 1) Куда нажимать

- Страница Books Sync: [/admin/ebookslv/books/](/admin/ebookslv/books/)
- Настройки Mozello (API-ключ, webhook): [/admin/ebookslv/mozello/](/admin/ebookslv/mozello/)

Кнопки на странице Books Sync:

- **Перезагрузить книги Calibre**
- **Загрузить товары Mozello**
- **Синхронизировать цены из Mozello**
- **Отправить цены в Mozello**
- **Экспортировать всё в магазин**
- **Удалить** (для отдельного продукта)

---

## 2) Частые задачи

### 2.1 Добавить новую платную книгу

1. Добавьте книгу в **Calibre** и заполните данные (название, обложка, описание, язык).
2. Установите **цену** в Calibre (поле `mz_price`).
   - Цена должна быть **больше 0**.
3. Откройте [/admin/ebookslv/books/](/admin/ebookslv/books/).
4. Нажмите **Перезагрузить книги Calibre**.
5. Выгрузите в Mozello:
   - Нажмите **Экспортировать всё в магазин**.
6. Проверьте корректность выгрузки:
   - Используйте кнопки **LV / RU / EN**, чтобы открыть страницу продукта в Mozello.

### 2.2 Изменить цену (Calibre → Mozello)

Используйте этот метод, если Calibre должен быть источником окончательной цены.

1. Измените цену в Calibre (`mz_price`).
2. Откройте [/admin/ebookslv/books/](/admin/ebookslv/books/).
3. Нажмите **Отправить цены в Mozello**.

### 2.3 Изменить цену (Mozello → Calibre)

Используйте этот метод, если цена была изменена напрямую в Mozello.

1. Откройте [/admin/ebookslv/books/](/admin/ebookslv/books/).
2. Нажмите **Синхронизировать цены из Mozello**.

### 2.4 Сделать платную книгу бесплатной

1. В Calibre установите цену (`mz_price`) в **0** или очистите поле.
2. Решите, должен ли продукт остаться в Mozello:
   - Если больше не нужно продавать: на [/admin/ebookslv/books/](/admin/ebookslv/books/) нажмите **Удалить** для этого продукта.

### 2.5 Удалить продукт Mozello

1. Откройте [/admin/ebookslv/books/](/admin/ebookslv/books/).
2. Нажмите **Удалить** для нужного продукта.

---

## 3) «Сироты» (продукт Mozello без соответствующей книги)

1. Нажмите **Загрузить товары Mozello**.
2. Строки с пометкой **ORPHAN** – это продукты, существующие в Mozello, но не соответствующие ни одной книге.
3. Обычно безопасно **Удалить** такой продукт и затем выгрузить правильную книгу из Calibre.

---

## 4) Быстрое устранение проблем

### 4.1 Экспорт не выполняется

- Проверьте настройки Mozello: [/admin/ebookslv/mozello/](/admin/ebookslv/mozello/).
- Попробуйте выполнить экспорт повторно, чтобы увидеть возможную ошибку.

### 4.2 Ссылки LV / RU / EN на продукт не открываются

- Нажмите **Загрузить товары Mozello** и повторите попытку.
- Если проблема сохраняется — повторно выполните экспорт.
